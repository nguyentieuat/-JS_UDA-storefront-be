Welcome to the Udacity Storefront

The following routes are supported:

    #### Need to register user by `/registry`[POST] to get token for next use, or keep token for next use
    
    #### Users
    - Index `/users` [GET] [token required]
    - Create `/users/` [POST] [token required]
    - Read `/users/:id` [GET] [token required]
    - Update `/users/:id` [PUT] [token required]
    - Delete `/users/:id` [DELETE] [token required]
    
    #### Products
    - Index `/products` [GET] [token required]
    - Create `/products` [POST] [token required]
    - Read `/products/:id` [GET] [token required]
    - Read `/products/cat/:category` [GET] [token required] (get product by category)
    - Delete `/products/:id` [DELETE] [token required]

    #### Orders
    - Index `/orders/:user_id` [GET] [token required] (get order by user)
    - Index `/orders/current/:user_id` [GET] [token required] (get order neareast by user)
    - Index `/orders/active/:user_id` [GET] [token required] (get order status active by user)
    - Index `/orders/completed/:user_id` [GET] [token required] (get order status completed by user)
    - Update `/orders` [PUT] [token required] (update status order)
    - Delete `/orders/:id` [DELETE] [token required]
